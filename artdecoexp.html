Versión mobile

Este sitio conforma la entrega final del curso de UTN desarrollador web fullstack con React js.
Principalmente se fue desarrollando en capas, desde la maquetación con html y css. Posteriormente incorporamos el framework React además de react router. En este punto comencé a idear los toques finales a nivel estético como filtros, sombras, efectos de iluminación y el rediseño del sitio en mobile basandome en el enfoque de "minimal changes".
Luego decidí agregar tanto el modelo 3d como el reproductor, utilizando todo el conocimiento disponible en javascript como las funcionalidades de los Hooks para estados o pasar datos de los padres a los componentes hijos. Por petición de los lineamientos del proyecto, tuve que darle un enfoque más comercial (también para demostrar ciertos conocimientos aprendidos en el curso) y de esta forma desarrollé el formulario, una base de datos en Sql y un apartado backend en NodeJs que aprovecha el entorno de ExpressJs junto a las dependencias disponibles como nodemailer, nodemon, express-session, etc.
En los ultimos pasos, se aplicó un enfoque MVC para poder administrar la base de datos con una app de Administrador. El deployment fue generado en el servicio de Heroku y versionado a través de git.

El objeto del proyecto desde un principio no solo fue demostrar conocimiento, sino crear una experiencia audiovisual con las herramientas que poseía en ese momento
