Sitio web para la empresa Transporte Servientrega SRL. El proyecto consistió de diferentes etapas entre las cuales se puede mencionar la creación del logo, la elección de una paleta de colores acorde a la identidad visual que la empresa intenta representar, diseño y posterior deployment. El sitio cuenta con versionamiento establecido en git y github además de la implementación de un WebHook para una actualización automática de los cambios y correción de errores. Diseño íntegro del sitio con énfasis en responsive desing con el fin de que el sitio sea accesible y optimizado para todas las pantallas disponibles.
Finalmente se realizó la inclusión de etiquetas SEO para mejor posicionamiento en buscadores (cards para redes sociales, metatags, alt, uso de html semantico, atributos como itemscope itemtype e itemprop/name) y la optimización de imagenes para web.
